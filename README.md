## クソコード作成指針
- Collectionとパッケージを使わない
- できる限りクラスを作らない
- 作れるsetter/getterは全部作る
- ネストは深ければ深いほど良い
- 1つのメソッド内でできることを分割しない
- if-else if..は長ければ長い方が良い
- 変数名，メソッド名は適当につける
- 一行に付くドットの数の限界に挑戦したい（クラスを作らないと難しい．．．）
